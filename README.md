🔐 Firebase User Management Web App

This project is a simple User Management System built using HTML, JavaScript, and Firebase. It allows users to create, read, update, and delete (CRUD) user data such as email, password, and username.

🚀 Features             
 Add new users (Email, Password, Username)          
 retrive user data from database            
 Update existing user details          
 Delete users from database            
 Real-time database interaction using Firebase          

🛠️ Technologies Used           
Frontend: HTML, JavaScript        
Backend/Database: Firebase Realtime Database / Firestore          
Authentication (optional): Firebase Auth
